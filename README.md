# CodeMetrics

Внутренняя платформа, которая **визуализирует метрики разработки**: по репозиториям, разработчикам и всему, что можно превратить в понятные графики.

Источником данных выступает **SferaCode** — мы забираем оттуда статистику/события, приводим к удобному виду и показываем в UI.

---

## Немного лора

CodeMetrics мы собрали **вдвоём за 2 дня** на хакатоне **Т1 (Новосибирск)**.  
В итоге заняли **2 место** и забрали **130 000 ₽**.

Проект родился из простой идеи: метрики есть почти везде, но обычно они либо спрятаны, либо выглядят так, что смотреть больно. Мы сделали так, чтобы смотреть было нормально

---

## Что есть внутри

> Без лишней воды — это именно платформа визуализации.

- Дашборды по **репозиториям** (динамика/активность/сравнения)
- Сводки по **разработчикам** (вклад, активность, тренды)
- Визуализация «прочих метрик» (в зависимости от того, что отдаёт SferaCode)
- API + Swagger, чтобы быстро проверять данные и гонять запросы

---

## Как это работает (упрощённо)

1) Backend ходит в **SferaCode** и забирает метрики  
2) Кладёт/кеширует их в **PostgreSQL / Redis**  
3) Отдаёт всё это через API  
4) Frontend рисует графики и страницы
